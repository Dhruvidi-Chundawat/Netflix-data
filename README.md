# Netflix Exploratory Data Analysis

This project aims to explore the content library of Netflix to gain insights into
content distribution, growth patterns, genre popularity, and targeting audiences.

## Tools
- Python
- Pandas, NumPy
- Seaborn, Matplotlib

## Key Insights
- Movies are the dominant content type on Netflix
- High growth rate after 2015
- USA, India, and UK are top content-producing countries
- TV shows with a limited number of seasons are in demand

## Business Impact
The insights gained can be used for strategic business decisions on content
acquisition and expansion plans.
